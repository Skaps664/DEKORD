## 1. Google Search Console

### **What It Does:**
- Monitors your site's presence in Google Search results
- Tracks keywords, impressions, clicks
- Identifies indexing issues
- Submits sitemaps
- Monitors mobile usability

### **Setup Steps:**

#### **A. Add Property**

1. **Go to Google Search Console:**
   - Open: https://search.google.com/search-console
   - Sign in with your Google account

2. **Add Property:**
   - Click **"Add Property"**
   - Select **"URL prefix"**
   - Enter: `https://dekord.online`
   - Click **"Continue"**

#### **B. Verify Ownership**

**Method 1: HTML File Upload (Recommended)**

1. Download verification file (e.g., `google1234abcd.html`)
2. Add file to `/website/public/` folder
3. Commit and push to GitHub:
   ```bash
   cd /home/skaps/dekord/website
   # File will auto-deploy with Vercel
   git add public/google*.html
   git commit -m "Add Google Search Console verification"
   git push
   ```
4. Wait 2-3 minutes for Vercel deployment
5. Click **"Verify"** in Search Console

**Method 2: HTML Meta Tag (Alternative)**

1. Copy the meta tag (looks like: `<meta name="google-site-verification" content="...">`)
2. I'll add it to your `layout.tsx` - let me know if you want this method!

#### **C. Submit Sitemap**

1. After verification, go to **"Sitemaps"** in left sidebar
2. Enter sitemap URL: `https://dekord.online/sitemap.xml`
3. Click **"Submit"**
4. ‚úÖ Your sitemap is already generated by Next.js at `/app/sitemap.ts`

#### **D. Verify Sitemap Works**

Test in browser:
- Open: https://dekord.online/sitemap.xml
- Should see XML with all your pages (products, collections, blog posts)

#### **E. Monitor Performance**

After 2-3 days, check:
- **Performance** ‚Üí See search impressions & clicks
- **Coverage** ‚Üí See indexed pages
- **Enhancements** ‚Üí Check mobile usability

## 2. Google Analytics 4

### **What It Does:**
- Tracks visitor behavior (pages, events, conversions)
- User demographics & interests
- E-commerce tracking (revenue, transactions)
- Real-time visitor monitoring
- Custom reports & funnels

### **Setup Steps:**

#### **A. Create GA4 Property**

1. **Go to Google Analytics:**
   - Open: https://analytics.google.com
   - Sign in with your Google account

2. **Create Account** (if you don't have one):
   - Click **"Start measuring"**
   - Account name: `Dekord`
   - Check data sharing settings (optional)
   - Click **"Next"**

3. **Create Property:**
   - Property name: `Dekord Online`
   - Time zone: `(UTC+05:00) Islamabad, Karachi` (Pakistan)
   - Currency: `Pakistani Rupee (PKR)`
   - Click **"Next"**

4. **Business Details:**
   - Industry: `Retail & Consumer Goods` or `Home & Garden`
   - Business size: Choose appropriate size
   - Click **"Next"**

5. **Business Objectives:**
   - Select: **"Get baseline reports"** and **"Measure customer engagement"**
   - Click **"Create"**

6. **Accept Terms of Service**

#### **B. Set Up Data Stream**

1. **Select Platform:**
   - Click **"Web"**

2. **Add Stream Details:**
   - Website URL: `https://dekord.online`
   - Stream name: `Dekord Website`
   - Click **"Create stream"**

3. **Copy Measurement ID:**
   - You'll see: `G-XXXXXXXXXX`
   - Copy this ID (you'll need it)

#### **C. Add GA4 to Your Website**

I'll add the tracking code. Your Measurement ID will go in `.env.local`:

**Add to `.env.local`:**
```bash
# Google Analytics 4
NEXT_PUBLIC_GA_MEASUREMENT_ID=G-XXXXXXXXXX
```

Then I'll create the GA4 component. Let me know when you have your Measurement ID!

#### **D. Enable E-commerce Tracking**

In GA4:
1. Go to **Admin** (bottom left)
2. Click **"Data streams"** ‚Üí Select your stream
3. Scroll to **"Enhanced measurement"**
4. Toggle ON:
   - ‚úÖ Page views
   - ‚úÖ Scrolls
   - ‚úÖ Outbound clicks
   - ‚úÖ Site search
   - ‚úÖ Form interactions
5. Click **"E-commerce events"** ‚Üí Toggle ON

#### **E. Set Up Conversions**

1. Go to **Admin** ‚Üí **Events**
2. Click **"Create event"**
3. Create these custom events:
   - `purchase` (for completed orders)
   - `add_to_cart`
   - `begin_checkout`
   - `view_item`

I'll implement these events in the code!

## 6. Facebook Pixel

### **What It Does:**
- Tracks conversions from Facebook/Instagram ads
- Retargeting campaigns (show ads to site visitors)
- Custom audiences
- Lookalike audiences
- Measure ad performance

### **Setup Steps:**

#### **A. Create Facebook Pixel**

1. **Go to Events Manager:**
   - Open: https://business.facebook.com/events_manager2
   - Select your Business Manager account

2. **Create Pixel:**
   - Click **"Connect data sources"**
   - Select **"Web"**
   - Click **"Connect"**
   - Select **"Facebook Pixel"**
   - Click **"Connect"**

3. **Name Your Pixel:**
   - Pixel name: `Dekord Pixel`
   - Website URL: `https://dekord.online`
   - Click **"Continue"**

4. **Copy Pixel ID:**
   - You'll see a Pixel ID (e.g., `123456789012345`)
   - Keep this for later

#### **B. Add Pixel to Your Website**

**Good news!** You already have Facebook Pixel installed! 

Check your `.env.local`:
```bash
NEXT_PUBLIC_FACEBOOK_PIXEL_ID=25319178191018029
```

**Update with your NEW Pixel ID:**
1. Replace the old ID with your new Pixel ID
2. Save file
3. Restart dev server or redeploy

#### **C. Verify Pixel Installation**

1. **Install Facebook Pixel Helper:**
   - Chrome Extension: https://chrome.google.com/webstore/detail/facebook-pixel-helper/
   - Install and pin to toolbar

2. **Test Pixel:**
   - Visit: https://dekord.online
   - Click Pixel Helper icon
   - Should show: ‚úÖ Green checkmark + "PageView" event

3. **Check Events Manager:**
   - Go to Events Manager
   - Select your Pixel
   - Go to **"Test Events"** tab
   - Visit your site
   - Should see real-time events!

#### **D. Set Up Standard Events**

Your site already tracks these events (implemented in `facebook-pixel.tsx`):
- ‚úÖ `PageView` - Page loads
- ‚úÖ `ViewContent` - Product page views
- ‚úÖ `AddToCart` - Add to cart clicks
- ‚úÖ `InitiateCheckout` - Checkout started
- ‚úÖ `Purchase` - Order completed

#### **E. Verify Domain**

1. **Go to Events Manager:**
   - Click **"Settings"** ‚Üí **"Domains"**
   - Click **"Add"** ‚Üí Enter `dekord.online`

2. **Verify Domain:**
   
   **Method 1: Meta Tag (Easiest)**
   - Copy the meta tag
   - I'll add it to your `layout.tsx`

   **Method 2: DNS TXT Record**
   - Copy the TXT record
   - Add to your domain DNS settings

3. **Aggregate Events:**
   - After verification, prioritize events:
     1. Purchase
     2. InitiateCheckout
     3. AddToCart
     4. ViewContent
